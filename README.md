## Prerequisites

- NodeJs

## Installing

Clone this repo

```
git clone https://github.com/acedesigns/bcb-coin-prices.git
```

Change into the created directory

```
cd bcb-coin-prices
```
Install Dependencies
```
npm install
```

## Running The App

```$xslt
npm run nx -- serve api
```
Open another terminal and run the front-end

```$xslt
npm run nx -- serve bitcoin
```
